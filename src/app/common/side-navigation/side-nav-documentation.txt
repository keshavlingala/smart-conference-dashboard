Properties
------------------------------------------------------------------------------------------------------------------------------
@Input('heading'):string

The heading to be shown in the left side nav.
If it is not passed then no heading will be shown.
--------------------------------------------------------------------------------------------------------------------------------
@Input('showLogoutBtn):boolean

As the name suggests, if you do not want to show the logout button then pass it as false.
------------------------------------------------------------------------------------------------------------------------------
@Input('navItems'): (NavItem|GroupedNavItem)[]

It is list of nav items to be shown.
Some of the list items may be grouped
------------------------------------------------------------------------------------------------------------------------------
groupedNavItems:number[]

It contains the indexes of all the grouped nav items in the @Input('navItems').
------------------------------------------------------------------------------------------------------------------------------
@ViewChildren('expansionPanel') expansionPanels: QueryList<any>

It is an list of all the instances of MatExpansionPanel (It is an Angular material component)
MatExpansionPanel is used for grouping of nav items.
------------------------------------------------------------------------------------------------------------------------------
@ViewChild('side_nav') sideNav: ElementRef;

It is an reference to the left side nav native html element.
------------------------------------------------------------------------------------------------------------------------------
